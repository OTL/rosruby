ROS Subscriber

 subscriber of ROS topic. Please use proc block for callback method.
 See below usage.

Usage

  node = ROS::Node.new('/rosruby/sample_subscriber')
  node.subscribe('/chatter', Std_msgs::String) do |msg|
    puts "message come! = \'#{msg.data}\'"
  end

  while node.ok?
    node.spin_once
    sleep(1)
  end
